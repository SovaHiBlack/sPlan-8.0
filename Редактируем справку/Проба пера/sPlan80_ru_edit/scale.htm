Scale

Miscellaneous Functions ››

 

You can give each page in sPlan its own scale.

sPlan then automatically takes this scale into account for all coordinates, rulers, dimensions, etc.


The properties of a page are displayed in the properties panel on the right edge of the window if no other element is currently selected on the plan. You can also find the Scale section here:
