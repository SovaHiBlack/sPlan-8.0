Split Components

Components ››

 

You have the option of splitting individual components into their constituent parts. All properties of the component are lost in the process. After splitting a component, it is only available graphically on the drawing.


To split a component, first select the desired component. Now select the Split selected component command from the Components menu. You can also select and split several components at the same time.
