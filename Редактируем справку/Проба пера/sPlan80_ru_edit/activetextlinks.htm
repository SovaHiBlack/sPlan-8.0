Активные Текстовые Ссылки

Различные Функции ››


В sPlan вы можете преобразовывать обычные тексты в активные ссылки.

Существует 2 типа ссылок: внешние и внутренние:


Внешние ссылки:

Вы можете использовать это для создания ссылки на веб-сайт (например, "www.abacom-online.de") или для прямой ссылки на определенный файл (например, PDF-файл со страницей данных).


Внутренние ссылки:

С помощью sPlan вы можете создавать активные ссылки из одной точки чертежа в другую. Не имеет значения, находится ли ссылка на том же листе или на совершенно другом листе вашего проекта. Вы можете автоматически переходить по ссылкам простыми щелчками мыши. Таким образом, вы можете определять переходы к совершенно разным частям вашего проекта и, таким образом, создавать интерактивные чертежи.

Внутренняя ссылка:


Щелчок по кнопке"Выберите цель ..."отображает список всех возможных целей перехода для вашего проекта. Выберите желаемую цель перехода из этого списка. Если вы еще не утвердили какой-либо другой текст в качестве цели перехода, этот список изначально пуст.


С Удалить цель вы можете удалить определенную цель перехода.


С Выберите новую цель... вы можете выбрать новый пункт назначения перехода, даже если другой пункт назначения перехода уже определен.


С помощью опции "Опубликовать как цель" вы определяете, должен ли текст быть доступен в качестве цели перехода по другим ссылкам.

Если вы хотите удалить целевой параметр из текста, нажмите на кнопку "Прекратить публикацию как цели"



Текстовые элементы с активными ссылками помечаются автоматически:


 

Все возможные цели для перехода отмечены с левой стороны стрелкой указывающей внутрь.

Все ссылки на цель обозначаются с правой стороны стрелкой указывающей наружу.

Такая идентификация позволяет сразу определить, с какими текстовыми элементами связаны ссылки.


Переход по ссылке

Если вы наведете курсор мыши на активную ссылку, отобразится адресат этой ссылки.

Затем вы можете перейти по ссылке двойным щелчком мыши.


Если вы наведете курсор мыши на правая внешняя стрелка, появится небольшое окно с целевой ссылкой.

Дважды щелкнув по правой внешней стрелке, вы попадете к тексту, на который дана ссылка.



Если вы наведете курсор мыши налевая внутренняя стрелка, появится небольшое окно с текстом, связанным с этим текстом.

Дважды щелкнув по левой внутренней стрелке, вы попадете на ссылку, которая ведет к этому тексту (источнику).



Создание списка ссылок

В sPlan вы можете создать список ссылок. Этот список содержит все ссылки вместе с их названиями и страницами. Чтобы создать список ссылок, выберите пункт Создать список ссылок... в меню Функции

Однако обычно имеет смысл сравнить отображаемый текст со ссылкой.
