Align Elements

Drawing of circuit diagrams ››


You can use this function to automatically align several elements with one another.


First, select the elements that you want to align. At least 2 elements must be selected.

Then select the "Align" button from the toolbar:



A menu then appears in which you can select the desired orientation. You have various options here:



After you have chosen an alignment, all selected elements are aligned as required
