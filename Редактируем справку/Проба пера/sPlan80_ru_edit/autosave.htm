Autosave

General Settings ››

 

Here you can activate 2 additional security mechanisms that can protect against data loss in certain cases.


You call up the settings dialog from the main menu under Settings -> General settings...

Then select the entry Autosave from the left bar:

 

 

With the first option you can activate a function in which the currently open file is automatically saved in a further backup file at regular intervals. The backup file has the same file name as the original file, only the extension is ".bak" instead of ".spl8".

If for whatever reason the original file should be defective, rename the backup file to ".spl8" and try to open it.


The second option provides even more security against data loss. Every time a file is opened, a copy of the original file is automatically saved as a backup file. This backup file always has the status of the file when it was last opened. These backup files always have the name "Backup_of_" followed by the original name of the file.
