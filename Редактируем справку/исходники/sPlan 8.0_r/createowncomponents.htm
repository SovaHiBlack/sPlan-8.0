Создавайте собственные компоненты

Components ››

Есть несколько способов создать свои собственные компоненты. Первое, что вам нужно сделать, это следующее: следует ли создавать компонент с нуля или его лучше создать на основе существующего компонента?

 

Создать компонент с нуля

Если вы хотите создать компонент с нуля, щелкните библиотеку правой кнопкой мыши и выберите команду «Создать».

Когда вы выполняете эту команду, новый компонент добавляется прямо в нижнюю часть библиотеки. По умолчанию sPlan всегда создает новый компонент в виде квадрата со стороной 20 мм в качестве заполнителя. Свойства нового компонента отображаются в правом углу окна. Теперь вы можете изменить это здесь или вызвать Компонент-редактор.

 

Создать новый компонент из существующего компонента

Если вы хотите создать компонент из другого аналогичного компонента, щелкните правой кнопкой мыши компонент, который нужно скопировать в библиотеку, и выберите команду «Дублировать». Копия компонента появляется сразу же внизу как последний компонент на странице библиотеки. Свойства нового компонента отображаются в правом углу окна. Теперь вы можете изменить это здесь или вызвать Компонент-редактор.

 

Создать компонент из выбора

Есть третий способ создать новый компонент. Для этого просто выберите на чертеже элементы, из которых должен состоять новый компонент. Затем выберите команду «Создать компонент из выделенного» в меню «Компоненты». Компонент создается сразу из выбранных элементов.
