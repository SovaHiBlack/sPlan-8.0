Каталоги

General Settings ››

 

При работе с sPlan вы можете задать определенные спецификации для рабочих каталогов и других специальных каталогов.

 

Вы вызываете диалоговое окно настроек из главного меню в разделе Настройки -> Общие настройки ...

Затем выберите запись «Каталоги» на левой панели.

   

Рабочие каталоги

При необходимости вы можете указать здесь фиксированные каталоги для файловых диалогов.

Если вы оставите поля пустыми, sPlan запомнит последние использованные каталоги и использует их снова в следующий раз.

 

Фиксированные каталоги

При необходимости вы можете изменить здесь каталоги по умолчанию для шаблонов и форм.

При нажатии кнопки Установить по умолчанию эти каталоги сбрасываются в каталоги sPlan по умолчанию.
