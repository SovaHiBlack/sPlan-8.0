Управление с клавиатуры

Miscellaneous Functions ››

Вы также можете использовать клавиатуру для доступа ко многим функциям sPlan, если это необходимо.

 

Доступ к часто необходимым функциям, таким как копирование, вставка, дублирование, удаление и т. Д., Можно получить с помощью специальных комбинаций клавиш. Эти комбинации клавиш отображаются рядом с пунктом меню в главном меню sPlan.

 

Другими функциями можно управлять с клавиатуры:

 

Режимы рисования

На вертикальной панели инструментов есть горячая клавиша для каждого режима рисования. Таким образом, вы можете просто выбрать соответствующий режим рисования с помощью клавиатуры вместо мыши.

Стандартно предустановлены следующие горячие клавиши:

 

Изменить = ESC

Линия = L

Подключение = P

Прямоугольник = R

Круг = C

Многоугольник = O

Безье = U

Произвольная линия = H

Особые формы = S

Размеры = D

Текст = T

TextRect = E

Растровое изображение = F

Масштаб = Z

Измерение = M

 

в общие настройки при необходимости вы можете сами определить горячие клавиши.

 

Перемещайте элементы с помощью клавиш со стрелками

Выбранные объекты на чертеже также можно перемещать с помощью клавиш со стрелками на клавиатуре. Элементы смещаются на одну точку сетки в желаемом направлении. Если вы также нажмете клавишу CTRL, вы можете переключить сетку на мелкую (0,1 мм) и настроить ее с помощью клавиатуры.

 

Выберите элементы с помощью клавиши TAB

С помощью клавиши TAB вы можете выбирать отдельные элементы чертежа один за другим. Каждое нажатие клавиши TAB выбирает следующий элемент. Если вы также нажмете клавишу SHIFT, будет выбран предыдущий элемент вместо следующего.

 

Вызов редактора компонентов с помощью Alt-Return

Когда компонент выбран, вы можете вызвать Компонент-редактор с Alt-Return.

 

Масштабирование с помощью клавиатуры

F5 увеличивает масштаб всей страницы

F6 увеличивает масштаб всех элементов рисунка.

F7 увеличивает масштаб всех выбранных элементов чертежа.

+ увеличивает

- уменьшает масштаб

 

Перейдите по текстовой ссылке с помощью F8

С помощью функциональной клавиши F8 вы можете следить за активной текстовой ссылкой.

Для этого необходимо выделить текстовый элемент со ссылкой.
