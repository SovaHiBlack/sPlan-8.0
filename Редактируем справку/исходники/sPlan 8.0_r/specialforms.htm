Особые формы

Специальные формы просты, но часто требуются, а иногда и трудны для создания элементов.

Часто эти формы также могут использоваться в качестве вспомогательного средства для строительства. Например, вы можете использовать равносторонний многоугольник в качестве вспомогательного средства для создания точной круглой формы точек пайки. Для этого просто создайте многоугольник с нужным количеством углов. Теперь поместите точки пайки на каждый угол, а затем снова удалите многоугольник. Вы уже выровняли точки пайки точно по круглой форме.

 

 

Создать новую специальную форму

 

Выберите подходящий режим рисования.

 

Откроется всплывающее меню, и вы можете выбрать нужную специальную форму из списка.

С первыми четырьмя специальными формами также появляется диалоговое окно для установки специальных свойств:


С помощью многоугольника и звезды вы можете выбрать, должен ли элемент создаваться как линия или как многоугольник. Если элементу нужно присвоить цвет заливки, вы должны выбрать «как многоугольник».

 

После выбора специальной формы (и, возможно, подтверждения диалога с помощью OK) вы можете поместить выбранную специальную форму на чертеж с помощью мыши. Для этого щелкните мышью по чертежу, чтобы определить начальную точку. Теперь, все еще удерживая кнопку мыши нажатой, нарисуйте мышью рамку, чтобы определить ее размер. Когда вы отпускаете кнопку мыши, на чертеже создается особая форма. С помощью некоторых специальных форм (например, стрелок) вы можете влиять на форму, выбирая направление, в котором вы открываете рамку (влево / вправо или вверх / вниз).

 

Изменить существующую специальную форму

После создания специальной формы она создается из основных элементов sPlan (линия, многоугольник и т. Д.).

Таким образом, изменение этих элементов аналогично изменению этих простых базовых элементов.

Специальные свойства, которые были выбраны при создании особой формы, больше не доступны после создания и не могут быть изменены впоследствии. При необходимости специальную форму можно просто удалить и создать новую.
