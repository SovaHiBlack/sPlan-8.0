Новое в версии 8.0

Welcome to sPlan 8.0 ››

Для тех, кто переходит с версии 7.0, есть список всех основных нововведений в sPlan 8.0:

 

Новый графический движок

sPlan 8.0 теперь использует современный графический движок Windows (Direct2D). Это приводит к гораздо лучшему, а также значительно более быстрому выходу графики. Теперь вы можете масштабировать и плавно перемещаться даже по более сложным чертежам, что значительно упрощает работу с sPlan.

 

Поддержка мониторов с высоким разрешением (масштабирование с высоким разрешением)

sPlan 8.0 теперь напрямую поддерживает масштабирование на мониторах с высоким разрешением. sPlan принимает масштабирование, установленное в системе Windows, и масштабируется под него.

 

Панель свойств

Благодаря новой панели свойств на правом краю окна все свойства элементов теперь можно изменять или просматривать непосредственно в любое время. Больше нет необходимости вызывать диалоговое окно двойным щелчком. Любые изменения свойства также сразу видны на вашем чертеже. Таким образом вы работаете еще более эффективно и интуитивно.

 

Новая библиотека компонентов

Обновленная библиотека компонентов теперь многоуровневая (с подкатегориями). Это позволяет упорядочивать библиотеку более структурированным и понятным образом. Теперь можно создать резервную копию всей библиотеки или восстановить резервную копию одним нажатием кнопки.

 

Юникод

sPlan теперь поддерживает полный набор символов Unicode. Это означает, что sPlan может отображать все символы, такие как греческие буквы или специальные математические символы.

 

Цифровая запись

Спецификации размеров отдельных элементов, таких как круги или прямоугольники, или спецификации положения теперь также могут быть указаны численно непосредственно в свойствах, если это необходимо.

 

Списки новых компонентов

Списки компонентов теперь можно определять более точно. Кроме того, при желании списки компонентов можно вставить прямо на вашу страницу в виде таблицы.

 

Оглавление

При желании оглавление можно вставить прямо в ваш рисунок как своего рода таблицу.

 

Угловая привязка при рисовании линий

Эта функция позволяет линиям вставлять на место под углом 45 A при рисовании или редактировании. Это позволяет автоматически и интуитивно создавать точно выровненные соединения даже за пределами установленной сетки привязки.

 

Растровые функции

Разрешение (dpi) вставленных растровых изображений теперь также можно регулировать (уменьшать) непосредственно в sPlan. Разумное сокращение может сэкономить много места для хранения. Существует также обозреватель растровых изображений, с помощью которого вы можете перечислить все растровые изображения, вставленные в проект. Это позволяет отследить ненужные «пожиратели памяти».

 

Текстовые ссылки / ссылки на страницы

Теперь есть несколько новых переменных для автоматических обозначений, доступных для создания текстовых ссылок или ссылок на страницы.

 

Линия преобразования <> многоугольник

Теперь каждую линию можно преобразовать в многоугольник, а каждый многоугольник - в линию. Поначалу это звучит непривлекательно, но это очень ценный помощник при рисовании.

 

Вылупление

sPlan 8.0 теперь использует собственную технику штриховки для заливки элементов. Штриховка теперь также может быть точно определена с помощью ширины линии и межстрочного интервала. Эта новая техника также устраняет любые проблемы с штриховкой при печати.

 

Произвольные линии

Новый режим рисования позволяет легко рисовать линии от руки.

 

Список страниц

Новый дополнительный вертикальный список страниц упрощает навигацию по страницам, особенно для очень больших проектов с более чем 20 страницами.

 

Горячие клавиши для режимов рисования

С помощью (редактируемых) горячих клавиш для отдельных режимов рисования вы можете переключаться между режимами еще быстрее, если это необходимо.

 

Полноэкранный режим

В полноэкранном режиме вы можете отображать свой рисунок в полном формате без каких-либо оконных рамок или других элементов на всем мониторе. Это может быть особенно полезно для презентаций.
