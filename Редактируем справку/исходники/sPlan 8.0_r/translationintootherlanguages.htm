Перевод на другие языки

Miscellaneous Functions ››


Большинство текстов, используемых в sPlan, сохраняются в файле под названием «Translations.txt».

Этот файл находится в папке «C: \ ProgramData \ sPlan80 \».

Название этой папки может отличаться в зависимости от версии Windows.

 

Этот файл содержит все тексты в следующем формате:


1100;Bearbeiten;Edit;Editer

1101;Rückgängig;Undo;Retour en arrière

1102;Wiederherstellen;Redo;Retour en avant

...


·          Каждая строка определяет ровно один текст

·          Каждая строка начинается с идентификационного номера, который sPlan использует для идентификации текста.

·          Далее следуют 3 текста, разделенных ";"

·          Первый текст используется для выбора языка НЕМЕЦКИЙ

·          Второй текст используется для выбора языка АНГЛИЙСКИЙ

·          Третий текст используется для выбора языка ФРАНЦУЗСКИЙ

 

Если вы хотите ввести новый текст, вы можете просто добавить желаемый текст в качестве четвертого текста:

 

1100;Bearbeiten;Edit;Editer;Мой собственный текст


·         Обратите внимание, что ваш собственный текст должен быть разделен знаком ";" из последнего текста

·          Как только станет доступен четвертый текст, sPlan всегда будет использовать его, независимо от выбранного языка.

·          Таким образом, вы можете полностью перевести sPlan на новый язык.

·          Обратите внимание, что изменения в этом файле вступят в силу только после перезапуска sPlan.

 

Вы также можете просто удалить файл «Translations.txt».

При следующем запуске sPlan этот файл автоматически создается снова с 3 стандартными языками

(немецкий, английский, французский).

 

В этом файле сохраняются только те ИЗМЕНЕНИЯ, которые вы хотите.

Таким образом, вы можете просто удалить этот файл в любое время, чтобы повторно активировать перевод sPlan по умолчанию.
