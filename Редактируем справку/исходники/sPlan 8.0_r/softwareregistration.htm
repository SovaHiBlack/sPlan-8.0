Регистрация программного обеспечения

Miscellaneous Functions ››

 

This option has nothing to do with the actual function of the software. You can use it to register your software directly with ABACOM, the manufacturer of this software. We will then automatically notify you of updates and new versions.


You can register in the main menu under Help -> Software registration...

We recommend registering by email.


Note:

If you obtained the software directly from ABACOM, this separate registration is not necessary.
