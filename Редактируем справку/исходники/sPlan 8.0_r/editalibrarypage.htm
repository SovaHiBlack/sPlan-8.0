Редактировать страницу библиотеки

The Component Library ››

Вы можете спроектировать библиотеку компонентов по своему желанию. Вы получите все необходимые команды, щелкнув правой кнопкой мыши компонент в библиотеке:

 

Характеристики

Это вызывает свойства компонента текущего выбранного компонента.

Кроме того, вы можете просто дважды щелкнуть нужный компонент.

 

Свойства (все)

Это вызывает свойства компонента для всех компонентов на текущей странице библиотеки.

Внесенные здесь изменения применяются ко всем компонентам на текущей странице библиотеки. Так что будьте здесь осторожны.

 

Новый

Когда вы выполняете эту команду, новый компонент добавляется прямо в нижнюю часть библиотеки. По умолчанию sPlan всегда создает новый компонент в виде квадрата со стороной 20 мм в качестве заполнителя. Свойства нового компонента отображаются в правом углу окна. Теперь вы можете изменить это здесь или вызвать редактор компонентов.

 

Дубликат

Если вы хотите создать компонент из другого аналогичного компонента, щелкните правой кнопкой мыши компонент, который нужно скопировать в библиотеку, и выберите команду «Дублировать». Копия компонента появляется сразу же внизу как последний компонент на странице библиотеки. Свойства нового компонента отображаются в правом углу окна. Теперь вы можете изменить это здесь или вызвать редактор компонентов.

 

Удалить...

Это удалит выбранный компонент со страницы библиотеки.

 

Страница библиотеки ->

 

Новый...

Это создает новую пустую страницу библиотеки.

После выполнения этой команды введите желаемое имя для новой страницы. например, «резисторы». При присвоении имен убедитесь, что страницы библиотеки отсортированы по алфавиту. Вы также можете использовать имя для определения позиции страницы в списке.

 

Копировать ...

Таким образом вы копируете полную страницу библиотеки.

Здесь также создается новая страница библиотеки, как с командой «Страница библиотеки -> Создать». Однако новая страница не пуста, а содержит копию всех компонентов текущей выбранной страницы библиотеки.

 

Переименовать ...

С его помощью вы можете переименовать текущую страницу библиотеки.

При присвоении имен убедитесь, что отдельные страницы затем отсортированы в алфавитном порядке. Вы также можете использовать имя для определения позиции страницы в списке.

 

Удалить...

Это удалит текущую страницу библиотеки.

После подтверждения секретного вопроса страница безвозвратно удаляется из библиотеки.

 

Пустой...

Это удалит все компоненты с текущей страницы библиотеки.

После подтверждения секретного вопроса страница не удаляется, как в случае «Страница библиотеки -> удалить ...», но остается доступной, но пустой.

 

Копировать на страницу ->

 

Только эта страница библиотеки

Создается новая страница, и все компоненты текущей страницы библиотеки автоматически помещаются на эту новую страницу.

 

Все страницы библиотеки этого уровня

Для каждой страницы библиотеки создается новая страница, и все компоненты соответствующей страницы библиотеки автоматически помещаются на эту новую страницу.

 

Дополнительно ->

 

Создать резервную копию ...

Это создаст резервную копию вашей библиотеки в виде ZIP-файла.

 

Восстановление резервной копии...

Вы можете использовать это для восстановления ранее созданной резервной копии.

 

Обновить библиотеку

Этим вы заставляете читать библиотеку заново. Это особенно необходимо, если вы внесли изменения на уровне файлов в каталоге библиотеки. Это избавляет вас от необходимости перезапускать sPlan.

 

Исследователь ...

Это вызывает Windows Explorer с текущим каталогом библиотеки.

Эта функция предназначена только для опытных пользователей для работы непосредственно с файлами библиотеки (* .LIB). Здесь эти файлы можно копировать, удалять, переименовывать или пересортировывать на уровне файлов.

 

Сортировка компонентов на странице

Чтобы изменить положение компонента, просто перетащите его в желаемое положение. Просто щелкните компонент, удерживайте кнопку мыши, перетащите его в нужное место в библиотеке, а затем снова отпустите кнопку мыши. Положение, в которое вставлен перетаскиваемый компонент, при перетаскивании отображается красным квадратом.

 

Изменить порядок или сгруппировать страницы библиотеки

Библиотека в sPlan имеет структуру каталогов. Файлы библиотеки можно сортировать и группировать по подкаталогам. Однако эта функция доступна напрямую только на уровне файлов Windows. С помощью функции Extras -> Explorer (как описано выше) вы можете открыть окно каталога Windows с текущим каталогом библиотеки. Здесь вы найдете все файлы LIB, отсортированные в соответствующих подкаталогах. Опытные пользователи могут создавать здесь свои собственные подкаталоги на уровне файлов, копировать файлы LIB и т. Д., Чтобы адаптировать структуру каталогов библиотеки к вашим требованиям.

 

Копирование компонентов с одной страницы библиотеки на другую

Чтобы скопировать компонент с одной страницы библиотеки на другую, действуйте следующим образом:

 

·          Сначала выберите страницу библиотеки, содержащую соответствующий компонент.

·          Перетащите этот компонент на свою схему.

·          Теперь выберите страницу библиотеки, на которую нужно скопировать компонент.

·          Теперь щелкните правой кнопкой мыши компонент на чертеже и выберите команду Копировать выбранный компонент (ы) в библиотеку.

 

Совет:

С помощью последней команды вы можете скопировать не только 1 компонент, но и любое количество компонентов из чертежа в библиотеку одним махом.
