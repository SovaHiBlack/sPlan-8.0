Что такое компонент?

Components ››

Компонент на самом деле представляет собой особую группу нарисованных элементов. Например, компонент «резистор» в простейшем случае состоит только из нарисованного прямоугольника с двумя линиями на концах, обозначающими соединения. Когда эти элементы чертежа затем объединяются в компонент, происходит следующее:

 

·          Автоматически добавляются 2 новых текстовых элемента (идентификатор и значение)

·          Элементы чертежа и текстовые элементы сгруппированы.

 

По определению, каждый компонент имеет два идентификатора текстовых полей и значение. В нашем примере идентификатором может быть, например, «R1» и значение «1k». Эти тексты могут быть произвольно определены пользователем.

 

Таким образом, отдельные компоненты различаются «только» по внешнему виду. В принципе, все они имеют одинаковую структуру.

 

Программа sPlan также предлагает несколько специальных опций для компонентов, которые делают работу с ними проще и гибче:

 

·          Идентификатор текстового элемента при желании можно скрыть

·          Текстовый элемент Value при желании можно скрыть

·          Вы можете автоматически пронумеровать (увеличить) идентификатор

·          Значение может запрашиваться автоматически при перетаскивании компонента на чертеж.

·          Вы можете определить, должен ли компонент быть включен в список компонентов или нет

·          Вы можете указать дополнительные тексты для компонента при использовании в списке компонентов

·          Вы можете ввести заголовок для компонента в библиотеке

 

Библиотека не обязательно должна содержать только компоненты, как определено выше. Также возможно добавить символ в библиотеку в виде клипа. Клип - это не компонент, а просто группа элементов рисунка. Такой зажим может быть любым символом, графикой или всей схемной частью. Воображение не знает границ. (Кто сказал, что sPlan можно использовать только для рисования цепей?)
