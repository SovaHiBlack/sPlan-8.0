Главное окно

Welcome to sPlan 8.0 ››

Основные окна sPlan состоят из разных областей:

 

 

Панель инструментов расположена под главным меню. Здесь вы можете вызвать многие функции программы одним щелчком мыши. Если на некоторое время навести указатель мыши на кнопку, в позиции мыши появится короткий текст для кнопки панели инструментов.

 

Библиотека компонентов со светло-синим фоном расположена на левом краю окна. Здесь вы можете выбрать отдельные страницы библиотеки и перетащить компоненты на чертеж. Вы можете определить ширину библиотеки компонентов с помощью мыши. Для этого наведите указатель мыши на правую границу библиотеки. Указатель мыши теперь превратится в двойную стрелку. Теперь вы можете определить ширину библиотеки, удерживая кнопку мыши. Непосредственно под библиотекой вы найдете несколько кнопок, с помощью которых вы можете влиять на отображение библиотеки. Здесь вы можете, например, установить количество столбцов или скрыть названия компонентов.

 

Панель инструментов режима для отдельных режимов рисования расположена рядом с библиотекой компонентов.

 

Справа от него находится светло-желтая область рисования. Используйте это, чтобы создать свой рисунок.

 

Панель свойств расположена справа от области рисования. Здесь вы можете напрямую изменять свойства отдельных элементов или страницы.

 

Список вкладок для страниц расположен под областью рисования. Вот «вкладки» для отдельных страниц чертежа. Щелчка на соответствующей вкладке достаточно, чтобы отобразить соответствующую страницу.
