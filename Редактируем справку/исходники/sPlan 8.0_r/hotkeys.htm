Горячие клавиши

General Settings ››

Здесь вы можете определить горячие клавиши для отдельных режимов инструмента в sPlan. Каждой кнопке на вертикальной панели инструментов может быть назначена собственная кнопка. Затем вы можете выбрать соответствующий режим инструмента с помощью клавиатуры вместо мыши.

 

Вы вызываете диалоговое окно настроек из главного меню в разделе Настройки -> Общие настройки ...

Затем выберите запись Горячие клавиши на левой панели.
