Многостраничные проекты

Drawing of circuit diagrams ››

 

Проект может состоять из нескольких отдельных страниц в sPlan. Например, имеет смысл разделить большой рисунок, если это возможно, на несколько небольших рисунков, которые затем помещаются на одну страницу. Например, схему сложного аудиоусилителя можно разделить на следующие отдельные схемы:

 

·          Адаптер питания

·          Уровень входа

·          регулировка тембра

·          усилитель мощности

 

 

Затем все 4 отдельные страницы будут сохранены в файле sPlan, например, под названием «Amplifier.spl8».

 

Вы можете создать любое количество отдельных страниц в проекте (то есть в одном файле). Каждая страница имеет свое название и соответствующий рисунок. У каждой страницы есть своя «вкладка» с названием страницы внизу области рисования. Затем вы можете переключиться на соответствующую страницу, щелкнув соответствующую вкладку.

Вы также можете определить порядок страниц впоследствии, например, чтобы отсортировать отдельные страницы по приоритетам или подобным образом.

Вы можете сохранять и загружать отдельные страницы из чертежа отдельно. Это дает вам возможность переносить определенные страницы из сложной схемы в другие проекты.

 

«Список вкладок» под областью рисования имеет собственное локальное меню (контекстное меню). Вы можете сделать это, наведя указатель мыши на этот список вкладок и щелкнув правой кнопкой мыши один раз. Локальное меню обеспечивает быстрый и легкий доступ ко многим командам:

 

Свойства страницы ...

Непосредственно вызывает свойства страницы на панели свойств на правом краю окна. Смотрите также настройки страницы.

 

Создать новую страницу ...

Это позволяет вам добавить новую страницу. Вы можете свободно определять положение новой страницы в небольшом диалоге:

Копировать страницу ...

С его помощью вы можете скопировать страницу в проекте. Вы можете свободно определять положение и количество копий в небольшом диалоге:

 

Удалить страницы ...

Это удаляет страницу из проекта. Если вы подтвердите следующий контрольный вопрос, страница будет безвозвратно удалена.

Кстати, если у вас в проекте всего одна страница, вы не можете ее удалить. Каждый проект должен состоять как минимум из одной страницы.

 

Сортировать страницы ...

Если вы хотите изменить порядок отдельных страниц, вы можете использовать команду «Сортировать страницы ...». Здесь вы можете легко повторно отсортировать отдельные страницы в списке с помощью перетаскивания.

 

Загрузить страницу из файла / Сохранить страницу в файл

Вы не найдете эти две команды в локальном контекстном меню панели страниц, а только в главном меню в разделах Страница -> Загрузить страницу из файла ... и Страница -> Сохранить страницу в файл ...

Если вы хотите, чтобы одна страница была в другом файле, вы можете сохранить ее отдельно. Затем вы можете позже перезагрузить страницу из любого другого файла проекта. Сохраненные страницы имеют расширение *.BLT.

 Перейти на страницу ...

Навигация по списку вкладок страниц иногда может быть утомительной, особенно в проектах с большим количеством страниц. Чтобы быстро перейти на определенную страницу, просто щелкните эту стрелку и выберите страницу из списка страниц.

 

 Добавить страницу быстро

С его помощью вы можете быстро и легко добавить новую страницу, минуя лишний диалог.

 

Вертикальный список страниц

В частности, для крупных проектов с большим количеством страниц вертикальный список страниц может быть более четким, чем обычное расположение страниц в списке вкладок под планом. Вверху / справа на панели инструментов есть кнопка для переключения между режимами списка страниц:


С его помощью вы можете легко переключаться между обычным списком вкладок страниц и вертикальным списком страниц.
