Активные текстовые ссылки

Miscellaneous Functions ››

В sPlan вы можете конвертировать обычные тексты в активные ссылки.

Есть 2 типа ссылок: внешние и внутренние:

 

Внешние ссылки:

Вы можете использовать это для создания ссылки на веб-сайт (например, «www.abacom-online.de») или ссылки непосредственно на конкретный файл (например, файл PDF со страницей данных).

 Ссылка создается в свойствах текста:

     Содержимое файла ТЕ550_4.3.1.jpg

Внутренняя ссылка:

С помощью sPlan вы можете создавать активные ссылки из одной точки чертежа в другую. Не имеет значения, находится ли ссылка на той же странице или на совершенно другой странице вашего проекта. Вы можете автоматически переходить по ссылкам простым щелчком мыши. Здесь вы можете определять переходы к совершенно разным частям вашего проекта и таким образом создавать интерактивные рисунки.

 

Щелчок по кнопке «Выбрать цель ...» показывает список всех возможных целей перехода для вашего проекта. Выберите желаемую точку перехода из этого списка. Если вы еще не одобрили какой-либо другой текст в качестве цели перехода, этот список изначально пуст.

 

С помощью Удалить цель вы можете удалить определенную цель прыжка.

 

С помощью Select new target ... вы можете выбрать новую цель перехода, даже если другая цель перехода уже была определена.

 

С помощью параметра «Опубликовать как цель» вы определяете, должен ли текст быть доступен в качестве цели перехода для других ссылок.

Если вы хотите удалить целевой параметр из текста, нажмите еще раз кнопку «Больше не публиковать как целевой».

 

Текстовые элементы с активными ссылками автоматически помечаются:

 

Все возможные цели прыжка получают стрелку, указывающую внутрь с левой стороны.

Все ссылки с целью отмечены стрелкой, указывающей наружу с правой стороны.

Эта идентификация позволяет сразу определить, какие текстовые элементы связаны.

 

Перейти по ссылке

Если вы наведите указатель мыши на активную ссылку, отобразится цель этой ссылки.

Затем вы можете выполнить ссылку двойным щелчком. 

Если вы наведете указатель мыши на правую внешнюю стрелку, появится небольшое окно с целью ссылки.

Двойным щелчком по правой внешней стрелке вы переходите к связанному тексту.

Если вы наведете указатель мыши на левую внутреннюю стрелку, появится небольшое окно с текстом, связанным с этим текстом.

Двойным щелчком по левой внутренней стрелке вы переходите по ссылке, которая связана с этим текстом (источником).

 

Создать список ссылок

В sPlan вы можете создать список ссылок. Этот список содержит все ссылки вместе с их названиями и страницами. Чтобы создать список ссылок, выберите пункт «Создать список ссылок ...» в меню «Функции».

 

Внешняя ссылка:

Здесь вы можете ввести внешнюю ссылку на веб-сайт (например, «www.abacom-online.de») или сослаться непосредственно на конкретный файл (например, файл PDF со страницей данных).

 

Ваш текст будет подчеркнут на плане, например, вот так:

Если вы дважды щелкните подчеркивание мышью, вы выполните эту внешнюю ссылку после запроса безопасности, и вы либо попадете на указанный веб-сайт, либо откроется соответствующий файл.

 

Обратите внимание, что текст, отображаемый на карте, не обязательно совпадает с текстом на внешней ссылке.

Однако обычно имеет смысл сравнить отображаемый текст со ссылкой.
