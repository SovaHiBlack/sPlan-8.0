Miscellaneous Functions ››

 

sPlan предлагает специальную функцию поиска для поиска компонентов или текстов и, при необходимости, для их автоматической замены. В частности, в случае несколько более крупных проектов с большим количеством страниц может быть сложно идентифицировать конкретный компонент или текст на чертеже.

 

Чтобы активировать поиск компонентов, выберите команду «Найти / заменить ...» в меню «Правка» или нажмите соответствующую кнопку (с выравниванием по правому краю на панели инструментов):

 

Панель поиска отобразится в правой части окна.

Вы можете снова скрыть панель поиска в любой момент, нажав еще раз.

 

Вверху вы можете выбрать, хотите ли вы искать компоненты или тексты:

 

   

Только поиск

 

Введите текст поиска для поиска компонентов в поля «Идентификатор» и / или «Значение» или для текстового поиска в поле «Текст для поиска».

 

Результаты поиска автоматически сортируются по страницам и отображаются в виде древовидной структуры.

Если щелкнуть результат поиска, элемент автоматически отобразится на чертеже. Если элемент находится на другой странице, страница также переключается автоматически. Таким образом, вы можете легко контролировать и просматривать все результаты поиска.

 

Поиск и повторное размещение

 

Во-первых, действуйте точно так же, как описано выше в разделе «Только поиск», чтобы отобразить результаты поиска.

 

Заменить один элемент

Если вы сейчас выберете результат, вы можете использовать кнопку «Элемент» ->, чтобы заменить текст в соответствующем элементе текстом, указанным в поле «Заменить». Это заменяет только текст в одном выбранном элементе.

 

Заменить все элементы страницы

Вы также можете заменить сразу все элементы страницы. Для этого выберите в результатах поиска вышестоящую страницу, а не отдельный элемент. Кнопка Элемент -> теперь становится кнопкой Полная страница ->. Теперь вы можете заменить сразу все элементы этой страницы.

 

Заменить все элементы на всех страницах

Вы также можете заменить сразу все элементы всех страниц. Для этого в результатах поиска выберите верхнюю запись Все страницы, а не отдельный элемент или страницу. Кнопка Элемент -> теперь становится кнопкой Все страницы ->. Теперь вы можете заменить все элементы на всех страницах (т.е. полный список результатов) сразу.

 

Параметры поиска

 

При поиске у вас все еще есть несколько вариантов, которые влияют на результат поиска:

 

Только элементы списка компонентов

Эта опция доступна только при поиске компонента. Если вы активируете это поле, будут учитываться только те компоненты, для которых активировано поле «Добавить в список компонентов». В любом случае это, вероятно, относится к большинству компонентов.

 

Деликатный случай

При поиске учитывается регистр или нет.

 

Решать

Если в тексте есть переменные, для поиска используется либо исходный текст с именами переменных («Это страница»), либо текст с разрешенными переменными («Это страница 1»). Например, если вы хотите искать имена переменных напрямую, вам придется отключить эту опцию.

 

Все страницы

Если эта опция активирована, поиск автоматически распространяется на все страницы проекта. В противном случае поиск будет производиться только на текущей странице.

 

Кнопка "Обновить"

Это позволяет в любой момент перезапустить поиск и обновить результаты.

Список результатов автоматически обновляется при каждом изменении в панели поиска. Однако, если был произведен поиск, а затем элементы были удалены или изменены в плане, может потребоваться ручное обновление.
