Список компонентов

Components ››

Важной функцией sPlan является создание списков компонентов. При необходимости они создаются программой полностью автоматически. Затем вы можете редактировать, распечатывать, загружать или сохранять список компонентов вручную по своему усмотрению.

 

Чтобы создать список компонентов, выберите команду «Создать список компонентов ...» в меню «Компоненты» или нажмите соответствующую кнопку на панели инструментов. Затем появляется следующее окно:

 

Здесь вы выбираете страницы, на которые должен распространяться список компонентов. После подтверждения этой информации появится список компонентов:

 

 

Вы можете редактировать список компонентов по своему усмотрению в окне списка компонентов. Здесь также есть следующие возможности:

 

Шрифт

Здесь вы можете установить шрифт и размер шрифта для списка компонентов.

 

Параметры

 

Данные файла

С помощью этой опции в список компонентов включается различная дополнительная информация:

 

·          Дата и время

·          Имя и папка файла проекта

·          Имена отдельных страниц для списка компонентов

 

Эти данные вставляются вверху списка компонентов.

 

Обратите внимание, что если вы измените эту опцию, список компонентов будет полностью воссоздан. Все изменения, внесенные вами вручную, будут потеряны.

 

Суммируйте равные компоненты

Если вы активируете эту опцию, все компоненты с одинаковым значением автоматически объединяются в одну запись в списке компонентов.

 

Что, например, выглядело так раньше ...

 

R1 = 1к

R2 = 1к

R3 = 1к

 

... затем становится:

 

R1, R2, R3 = 3 х 1k

 

Это значительно увеличивает ясность списка компонентов.

 

Обратите внимание, что если вы измените эту опцию, список компонентов будет полностью воссоздан. Все изменения, внесенные вами вручную, будут потеряны.

 

Дополнительный текст 1..4

Здесь вы можете показать или скрыть лишние столбцы для дополнительных текстов (Z1..Z4)

 

Вертикальные линии

Это позволяет добавлять вертикальные разделительные линии в список компонентов.

 

Горизонтальные линии

Это позволяет добавлять в список компонентов горизонтальные разделительные линии.

 

Сортировать группы

 

Количество

Если вы выберете эту опцию, наиболее часто представленная группа компонентов появится первой в списке компонентов. Все остальные группы также появляются в порядке их частоты.

 

По алфавиту

Если вы выберете эту опцию, группы компонентов будут отсортированы в алфавитном порядке по их идентификаторам.

 

Распечатать

 

Верхнее поле / левое поле

Перед печатью определите верхнее и левое поля печати, чтобы список компонентов не печатался прямо по краю.

 

Портрет / Пейзаж

Перед печатью определите размер бумаги.

 

Распечатать

Распечатывает список компонентов

 

Вставить на страницу ...

 

С помощью кнопки Вставить на страницу ... вы можете вставить список компонентов прямо на текущую страницу:  


 

Экспорт ...

 

С помощью кнопки Экспорт ... вы можете экспортировать список компонентов в виде текстового файла:

 

Буфер обмена

Эта функция копирует список компонентов в буфер обмена Windows. Отсюда его можно вставить любой другой программой (например, Word). Затем вы можете продолжить редактировать список компонентов в другой программе по своему желанию.

 

Сохранить...

С помощью этой кнопки вы можете сохранить текущий список компонентов. Список компонентов сохраняется в универсальном «формате RTF» и имеет окончание «.RTF». Большинство текстовых редакторов могут читать этот формат.

 

Открытым...

С помощью этой кнопки вы можете повторно открыть уже сохраненный список компонентов.
