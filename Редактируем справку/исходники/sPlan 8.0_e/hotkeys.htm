Hotkeys

General Settings ››


Here you can define hotkeys for the individual tool modes in sPlan. Each button in the vertical tool bar can be assigned its own button. You can then select the corresponding tool mode with the keyboard instead of the mouse.


You call up the settings dialog from the main menu under Settings -> General settings...

Then select the entry Hotkeys from the left bar.
