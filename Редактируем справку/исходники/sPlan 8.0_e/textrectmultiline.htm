TextRect (multiline)

Drawing of circuit diagrams ›› Drawing functions ››

 

 



Create a new TextRect

 

Select the appropriate mode switch.


Click with the mouse on the drawing in order to determine the starting point of the TextRect. Now, while keeping the mouse button pressed, draw a frame with the mouse to determine its size. When you release the mouse button, the TextRect is placed on your drawing. You can now directly create further TextRects or exit the TextRect mode with the right mouse button. Now you can directly adjust the properties (text, font, color, etc.) of the new TextRect.



Change an existing TextRect

If you select a TextRect on the drawing, you can use the sizers to change its size, shape and direction of rotation at any time.


       


Change properties of TextRects

If you select a TextRect, you can set its properties in the properties panel on the right edge of the window. You can also select several TextRects at the same time and thus change their properties together.



Tip:

Press the key combination <Ctrl> + <Ins> in the input field of the paragraph text to call up the special characters dialog.



Here you will find all characters that are available in the currently selected character set (Unicode), including those that cannot be accessed directly via the keyboard.


If you double-click on a character, it will automatically be included in the text.
